Koch Curve

Default Curve koch_curve(t, 4, 20)

The modified Koch Curve will move backwards half a step each time while changing it's color to any random rgb color. mod_koch_curve(t, 4, 20) 

      
        def mod_koch_curve(t, depth, size):
          if (depth == 1):
            t.color(((random.randrange(255)), (random.randrange(255)), (random.randrange(255))))
              t.fd(size)
              t.bk(size/2)
          else:
          mod_koch_curve(t, depth-1, size)
          t.lt(60)
          mod_koch_curve(t, depth-1, size)
          t.rt(120)
          mod_koch_curve(t, depth-1, size)
          t.lt(60)
          mod_koch_curve(t, depth-1, size)

Sierpinski Triangle

Default Sierpinski sierpinski(t, 5, 200)

The modified Sierpinski Triangle will change it's color to green and change it's pensize everytime given a range of 0-5. Assume that another function named "triangle" draws a equilateral triangle. mod_sierpinski(t, 5, 200) 

      
        def mod_sierpinski(t, depth, size, scale_factor=1):
            if depth == 1:
                triangle(t, size)
                t.pensize(random.randrange(5))
            else:
                t.color((0, 255, 0))
                mod_sierpinski(t, depth-1, size/2)
                t.fd(size/2)
                mod_sierpinski(t, depth-1, size/2)
                t.bk(size/2)
                t.lt(60)
                t.fd(size/2)
                t.rt(60)
                mod_sierpinski(t, depth-1, size/2)
                t.rt(120)
                t.fd(size/2)
                t.lt(120)

Tree

Default Tree tree(t, 4, 50, 20)

The modified Tree will change it's pensize based on the current depth+1. The branches will look thinner and thinner. It will also change one of it's branch color to a random shade of blue. I also changed the depth to 8. mod_tree(t, 8, 50, 20) 

      
        def mod_tree(t, depth, size, angle):
            if depth == 0:
                t.fd(size)
                t.bk(size)
                t.color((0, 0, (random.randrange(255))))
            else:
                t.pensize(depth+1)
                t.fd(size)
                t.rt(angle)
                mod_tree(t, depth-1, size, angle)
                t.pensize(depth+1)
                t.lt(2 * angle)
                mod_tree(t, depth-1, size, angle)
                t.pensize(depth+1)
                t.rt(angle)
                t.bk(size)
