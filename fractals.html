Koch Curve

Default Curve koch_curve(t, 4, 20) The modified Koch Curve will move backwards half a step each time while changing it's color to any random rgb color. mod_koch_curve(t, 4, 20) 
      
        def mod_koch_curve(t, depth, size):
          if (depth == 1):
            t.color(((random.randrange(255)), (random.randrange(255)), (random.randrange(255))))
              t.fd(size)
              t.bk(size/2)
          else:
          mod_koch_curve(t, depth-1, size)
          t.lt(60)
          mod_koch_curve(t, depth-1, size)
          t.rt(120)
          mod_koch_curve(t, depth-1, size)
          t.lt(60)
          mod_koch_curve(t, depth-1, size)

Sierpinski Triangle

Default Sierpinski sierpinski(michelangelo, 5, 200) The modified Sierpinski Triangle will change it's color to green and change it's pensize everytime given a range of 0-5. Assume that another function named "triangle" draws a equilateral triangle. mod_sierpinski(t, 5, 200) 
      
        def mod_sierpinski(t, depth, size, scale_factor=1):
            if depth == 1:
                triangle(t, size)
                t.pensize(random.randrange(5))
            else:
                t.color((0, 255, 0))
                mod_sierpinski(t, depth-1, size/2)
                t.fd(size/2)
                mod_sierpinski(t, depth-1, size/2)
                t.bk(size/2)
                t.lt(60)
                t.fd(size/2)
                t.rt(60)
                mod_sierpinski(t, depth-1, size/2)
                t.rt(120)
                t.fd(size/2)
                t.lt(120)
